# Million Dollar Mansion Casino

A browser-based casino tycoon game inspired by roguelike deckbuilders and economic simulators.

## Objective

Start with only $50.

Use your money to:

* Play casino games
* Buy powerful upgrades
* Haggle with shopkeepers
* Exploit changing market prices
* Unlock stronger items
* Reach $1,000,000
* Purchase the Mansion and win

---

## Features

### Casino Games

* Blackjack
* Poker
* Roulette
* Craps

### Dynamic Economy

Item prices change daily.

Possible market conditions:

* Bull Market
* Recession
* Inflation
* Casino Boom
* Luxury Craze

### Upgrade System

Collect powerful upgrades that improve your odds.

Item rarities:

* Common
* Uncommon
* Rare
* Epic
* Legendary
* Mythic

### Haggling

Attempt to negotiate item prices.

Success depends on:

* Reputation
* Negotiation skill
* Shopkeeper mood

### Prestige

After purchasing the Mansion:

* Reset progress
* Earn permanent bonuses
* Unlock exclusive content

---

## Controls

Use the navigation tabs to switch between:

* Casino
* Shop
* Inventory
* Statistics

Game progress is stored locally using:
